# Інструкція з налаштування домену loys.kiev.ua для Comet Assistant

## Мета завдання

Налаштувати кастомний домен **loys.kiev.ua** для сайту Адвокатського об'єднання "Яремчук і Седун", який розміщено на платформі Vercel за адресою https://loys-law-site.vercel.app

## Поточний статус

✅ **Виконано:**
- Сайт успішно розгорнуто на Vercel
- Домен loys.kiev.ua додано до проєкту Vercel
- Vercel готовий приймати трафік на цей домен

⚠️ **Потрібно зробити:**
- Налаштувати DNS записи на реєстраторі доменів nic.ua
- Дочекатися поширення DNS (10-30 хвилин)
- Перевірити роботу SSL сертифікату

## Крок 1: Вхід до панелі управління nic.ua

1. Перейдіть на сайт https://nic.ua
2. Увійдіть в особистий кабінет за допомогою облікових даних
3. Знайдіть домен **loys.kiev.ua** у списку ваших доменів
4. Відкрийте налаштування домену

**Важлива інформація:**
- Домен зареєстровано до: 2027-03-07 (залишилось достатньо часу)
- Реєстратор: nic.ua
- Тип домену: .kiev.ua (український географічний домен)

## Крок 2: Налаштування DNS записів

### Необхідні DNS записи:

#### А-запис (основний домен)
```
Тип: A
Ім'я: @ (або залиште порожнім, або вкажіть loys.kiev.ua)
Значення: 76.76.21.21
TTL: 3600 (або за замовчуванням)
```

**Пояснення:** Цей запис вказує, що домен loys.kiev.ua повинен направляти на IP-адресу серверів Vercel (76.76.21.21).

#### CNAME-запис (www піддомен) - опціонально
```
Тип: CNAME
Ім'я: www
Значення: loys.kiev.ua
TTL: 3600 (або за замовчуванням)
```

**Пояснення:** Цей запис забезпечує, щоб www.loys.kiev.ua також працював і редіректив на основний домен.

### Інструкція з налаштування на nic.ua:

1. У панелі управління доменом знайдіть розділ "DNS-сервери" або "Управління DNS"
2. Якщо домен використовує DNS-сервери nic.ua, перейдіть до редагування записів
3. **Видаліть або відключіть** старі A-записи, якщо вони є
4. **Додайте новий A-запис:**
   - Поле "Ім'я хосту" або "Host": залиште порожнім або введіть `@`
   - Поле "IP-адреса" або "Value": `76.76.21.21`
   - Поле "TTL": оберіть 3600 або залиште за замовчуванням
5. **Додайте CNAME-запис (опціонально):**
   - Поле "Ім'я хосту": `www`
   - Поле "Ціль" або "Target": `loys.kiev.ua`
6. **Збережіть зміни**

### Важливі нюанси для nic.ua:

- На nic.ua можливо потрібно буде перейти в розділ "Додаткові DNS-записи"
- Якщо домен використовує сторонні DNS-сервери (не nic.ua), змініть їх на DNS-сервери nic.ua:
  - `ns1.nic.ua`
  - `ns2.nic.ua`
  - `ns3.nic.ua`
- Після зміни DNS-серверів потрібно почекати 24-48 годин для повного поширення

## Крок 3: Перевірка конфігурації DNS

### Через 10-30 хвилин після налаштування:

1. Відкрийте термінал або командний рядок
2. Виконайте команду для перевірки A-запису:
   ```bash
   nslookup loys.kiev.ua
   ```

   **Очікуваний результат:**
   ```
   Server: ...
   Address: ...

   Name: loys.kiev.ua
   Address: 76.76.21.21
   ```

3. Альтернативна перевірка через онлайн-інструменти:
   - https://dnschecker.org - введіть loys.kiev.ua
   - https://www.whatsmydns.net - глобальна перевірка DNS

### Можливі проблеми та їх вирішення:

❌ **DNS все ще показує стару адресу**
- Рішення: Почекайте ще 15-30 хвилин, очистіть DNS кеш:
  ```bash
  # Windows
  ipconfig /flushdns

  # macOS
  sudo dscacheutil -flushcache

  # Linux
  sudo systemd-resolve --flush-caches
  ```

❌ **DNS не розв'язується взагалі**
- Рішення: Перевірте, чи правильно збережені DNS записи на nic.ua
- Перевірте, чи домен використовує DNS-сервери nic.ua

❌ **Показує помилку NXDOMAIN**
- Рішення: Можливо, A-запис не додано. Поверніться до панелі nic.ua

## Крок 4: Перевірка роботи сайту

### Після поширення DNS:

1. **Відкрийте браузер у режимі інкогніто** (щоб уникнути кешу)
2. Перейдіть на https://loys.kiev.ua
3. Переконайтеся, що:
   - ✅ Сайт відкривається
   - ✅ Показується головна сторінка Адвокатського об'єднання
   - ✅ В адресному рядку відображається замок (HTTPS працює)
   - ✅ Всі сторінки працюють (переходи по меню)
   - ✅ Чат-бот працює
   - ✅ Форма зворотного зв'язку працює

### Перевірка SSL сертифікату:

1. Клацніть на замок в адресному рядку браузера
2. Оберіть "Сертифікат" або "Connection is secure"
3. Переконайтеся, що:
   - Сертифікат виданий для loys.kiev.ua
   - Видавець: Let's Encrypt (через Vercel)
   - Термін дії: активний

**Примітка:** Vercel автоматично створює SSL сертифікат протягом декількох хвилин після того, як DNS записи почнуть працювати.

4. Переконайтеся, що:
   - ✅ loys.kiev.ua показує статус "Valid Configuration"

### Встановлення основного домену (опціонально):

1. У розділі Domains знайдіть loys-law-site.vercel.app
2. Клацніть три крапки (...)
3. Оберіть "Redirect to loys.kiev.ua"

## Типові помилки та їх вирішення

### Проблема 1: "Domain not verified"

**Симптоми:** Vercel показує, що домен не підтверджено

**Рішення:**
1. Переконайтеся, що A-запис `76.76.21.21` активний
2. Зачекайте 10-30 хвилин для поширення DNS
3. На Vercel натисніть "Refresh" або "Verify" біля домену

### Проблема 2: SSL сертифікат не створюється

**Симптоми:** Браузер показує попередження "Not Secure"

**Рішення:**
1. Це нормально протягом перших 5-15 хвилин
2. Vercel автоматично запитує SSL сертифікат від Let's Encrypt
3. Якщо через 30 хвилин проблема залишається:
   - Перевірте, що DNS записи правильні
   - Видаліть і додайте домен заново на Vercel

### Проблема 3: Сайт показує "Project not found"

**Симптоми:** Домен відкривається, але показує помилку Vercel

**Рішення:**
1. Переконайтеся, що домен додано до правильного проєкту на Vercel
2. Перевірте, що проєкт успішно задеплоєно (є активний deployment)

### Проблема 4: www.loys.kiev.ua не працює

**Симптоми:** Основний домен працює, але www - ні

**Рішення:**
1. Додайте CNAME запис на nic.ua (дивіться Крок 2)
2. АБО додайте www.loys.kiev.ua як окремий домен на Vercel
